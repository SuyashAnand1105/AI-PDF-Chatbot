⚠️ COMMON ERRORS (Don't Panic!):

1. "Port already in use"
   → This is NORMAL! Server is already running.
   → Just open http://localhost:5174 in your browser

2. "Network Error"
   → Make sure both terminal windows are still open
   → Refresh the page (F5)

3. "PDF processing failed"
   → Try a smaller PDF (under 10MB)
   → Wait at least 30 seconds

4. "AI error"
   → Wait a few seconds and try again
   → OpenRouter API might be temporarily busy
